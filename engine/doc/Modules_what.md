# What is modules?

* Modules are Novels with `export` folder
* Each module has folders with code in `export` folder

## Example
* File structure
```
novel/export/
	haxidenti_inventory/
		File1.ts
		File2.ts
		File3.ts
	haxidenti_messages/
		File1.ts
		File2.ts
		File3.ts
```


## Why they need?
* Allows to export functionality between novels
* Easy to export/import
